### Requirements

You will need these requirements:

- Node.js (v18 or higher) <a target='_blank' href='https://nodejs.org/en'>Download it here</a>
- npm / pnpm / yarn / bun (The package manager to install the library, we recommend **pnpm**) <a target='_blank' href='https://pnpm.io/'>Download it here</a>
- Mesalvo AWS CodeArtifact account

### Why do you need a Mesalvo AWS CodeArtifact account?

<Alert icon={TbInfoSquareRoundedFilled} color="primary">

### I have an AWS CodeArtifact account, what's next?

Great!, now you need to authenticate with the AWS CLI to download the package. Run the following commands in your terminal: <br/>
(You might need to install the AWS CLI first, check the [official documentation](https://docs.aws.amazon.com/cli/latest/userguide/install-cliv2.html) for more information)

> Option 1. Windows cmd

```bash
set AWS_ACCESS_KEY_ID=AK************4W
set AWS_SECRET_ACCESS_KEY=gA*********************************RW
set AWS_DEFAULT_REGION=***********
aws codeartifact login --tool npm --repository mesalvo-platform --domain mesalvo --domain-owner 905418322627 --region eu-central-1
FOR /F "tokens=*" %g IN ('aws codeartifact get-authorization-token --domain mesalvo --domain-owner 905418322627 --region eu-central-1 --query authorizationToken --output text') do (SET CODEARTIFACT_AUTH_TOKEN=%g)
```

> Option 2. Windows Powershell

```bash
# Set AWS credentials and default region
$env:AWS_ACCESS_KEY_ID = "AK************4W"
$env:AWS_SECRET_ACCESS_KEY = "gA*********************************RW"
$env:AWS_DEFAULT_REGION = "***********"
# Login to AWS CodeArtifact using npm tool
aws codeartifact login --tool npm --repository mesalvo-platform --domain mesalvo --domain-owner 905418322627 --region eu-central-1
# Get CodeArtifact authorization token and set it as an environment variable
$CodeArtifactAuthToken = aws codeartifact get-authorization-token --domain mesalvo --domain-owner 905418322627 --region eu-central-1 --query authorizationToken --output text
```

> Option 3. Linux / MacOS

Please check the [official AWS documentation](https://docs.aws.amazon.com/codeartifact/latest/ug/getting-started-cli.html) to authenticate in Linux or MacOS.

# I have the token and I'm ready to install the package

## Installation

To install the package, run the following command:

```bash
pnpm add @mesalvo/react-ui

# or with npm :
# npm install @mesalvo/react-ui

# or with yarn :
# yarn add @mesalvo/react-ui

```

## Usage

To use any component, import them in your project:

```jsx
import { ComponentName } from '@mesalvo/react-ui'
```
